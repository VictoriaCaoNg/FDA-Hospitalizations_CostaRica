1.3 Covariables rezagadas

Con el fin de explorar posibles efectos retardados, se ajustaron modelos que incorporan las covariables meteorológicas rezagadas una semana. Este análisis se llevó a cabo para cada una de las distribuciones consideradas (normal, Poisson, quasi-Poisson y binomial negativa).

lag_matrix <- function(x, k){
  cbind(
    matrix(NA, nrow(x), k),
    x[, 1:(ncol(x)-k)]
  )
}
tmax_lag1  <- lag_matrix(base$Tmax, 1)
tmin_lag1 <- lag_matrix(base$Tmin, 1)
precip_lag1  <- lag_matrix(base$precip, 1)
ampmax_lag1 <- lag_matrix(base$amp_max_mean, 1)
ampmin_lag1 <- lag_matrix(base$amp_min_mean, 1)
aod_lag1 <- lag_matrix(base$aerosol, 1)
hr_lag1 <- lag_matrix(base$HR, 1)

3 Modelo final

Para la selección del modelo final se consideró el rendimiento de cada uno, utilizando métricas como el error cuadrático medio (MSE, por sus siglas en inglés), el error absoluto medio (MAE), devianza y el coeficiente de determinación ajustado, además del grado en que cada modelo cumple con sus supuestos. En la tabla se puede observar dichas métricas para cada modelo.

# Normal
yhat_norm <- predict(mod_lag1_norm, type = "response")

Además, tomando en cuenta los supuestos, el modelo binomial negativa compensa la sobredispersión y cumple con los otros supuestos de independencia de los residuos. Otra ventaja es que permite el uso de criterios de información como el de Akaike y Bayes, a comparación del modelo quasi-poisson.

Tomando en cuenta estos criterios se escoge el modelo con distribución binomial negativa. Seguidamente, se procede con la selección de variables y número de componentes principales utilizados. La selección de variables se realizará mediante el Criterio de información de Akaike (AIC) y el Criterio de información de Bayes (BIC), donde se elimina una variable a la vez y se compara los valores de AIC y BIC entre los modelos.
